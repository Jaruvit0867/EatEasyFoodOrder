# คู่มือการติดตั้งสำหรับ Windows (Thai Manual)

หากคุณเพิ่งเริ่มใช้งานและยังไม่มีโปรแกรมพื้นฐาน ให้ทำตามขั้นตอนดังนี้ครับ

## 1. ติดตั้ง Python
โปรแกรมนี้จำเป็นสำหรับรันระบบหลังบ้าน (Backend)

1. **ดาวน์โหลด:** ไปที่ [Python Downloads](https://www.python.org/downloads/)
2. **ติดตั้ง:** เปิดไฟล์ที่โหลดมา
   > ⚠️ **สำคัญมาก:** ในหน้าแรกของการติดตั้ง **ต้องติ๊กถูก** ช่องที่เขียนว่า **"Add Python to PATH"** ด้านล่างสุด ก่อนกด Install No
3. **ตรวจสอบ:** เปิด Command Prompt (cmd) แล้วพิมพ์ `python --version` ถ้าขึ้นเลขเวอร์ชันแสดงว่าใช้ได้

## 2. ติดตั้ง Node.js
โปรแกรมนี้จำเป็นสำหรับหน้าเว็บ (Frontend)

1. **ดาวน์โหลด:** ไปที่ [Node.js Downloads](https://nodejs.org/en/download/) (แนะนำรุ่น LTS)
2. **ติดตั้ง:** กด Next ตามปกติจนเสร็จ
3. **ตรวจสอบ:** ใน cmd พิมพ์ `npm --version`

## 3. เริ่มใช้งาน EatEasy
เมื่อติดตั้งครบแล้ว:
1. ดับเบิ้ลคลิกไฟล์ `easy_setup.bat` เพื่อติดตั้งระบบ (ทำครั้งเดียว)
2. ดับเบิ้ลคลิกไฟล์ `easy_run.bat` เพื่อเปิดใช้งานได้เลย!
